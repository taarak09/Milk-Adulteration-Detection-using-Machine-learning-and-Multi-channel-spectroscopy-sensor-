# 🥛 Milk Adulteration Detection using ML & Spectroscopy

## 📖 Overview
This project detects milk adulteration using multi-channel spectroscopy sensors and a machine learning model deployed on Arduino.

## 🚀 Features
- Real-time milk quality detection
- Multi-channel spectroscopy sensor data
- pH-based feature integration
- ML model deployed on embedded system

## 🧠 Model Details
- Random Forest Model
- 45 Trees
- 42 Features (spectral + pH)
- 9 Output Classes

## ⚙️ Hardware
- Spectroscopy Sensor
- pH Sensor
- Arduino

## 📂 Files
- `Milk_scanner.ino` → Arduino code
- `model.h` → ML model (converted to C)

## 🔄 Workflow
1. Sensor collects data
2. Features are computed
3. Model predicts milk quality
4. Output is displayed

## 🎯 Real-World Impact
Can be used in dairy supply chains to detect milk adulteration in real-time.
